# Reverse Tower Defense

## App Store Description
反转经典塔防！你不再是守护者，而是入侵的怪物。选择快速跑者、坦克或隐身忍者，冲破层层塔防抵达终点。每关都有独特布局，考验你的策略选择。你能通关全部 10 关吗？

## Unique Selling Point
市面上第一款"反转视角"塔防游戏 — 你是怪物，塔是敌人。在 App Store 搜索 "reverse tower defense" 几乎没有精品竞品。

## Core Mechanic
- 选择三种怪物类型之一（速度型/坦克型/平衡型）
- 沿着预设路径前进，躲避或硬抗塔的攻击
- 到达终点即胜利，血量归零即失败
- 每关塔的数量、位置、火力都不同

## Content
- **Levels:** 10 个精心设计的关卡
- **Difficulty:** 从简单教程到地狱难度递增
- **Monster Types:** 3 种怪物，各有特色
- **Estimated playtime:** 每局 30-90 秒，总流程 15-30 分钟

## Age Rating
**9+** — 策略游戏，无暴力内容，适合青少年及以上。

## Technical Highlights
- React Native + Expo 52 + TypeScript
- react-native-reanimated 流畅动画
- expo-linear-gradient 精美渐变背景
- expo-haptics 触感反馈
- AsyncStorage 进度保存
- 完全离线，无网络依赖
